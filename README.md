# golang-practice

A personal workspace for Go programming practice and experimentation.

## Contents

- **Go with TDD** - A comprehensive journey through Go using Test-Driven Development covering modules like concurrency, pointers, context, reflection, and more.
- **CLI w Golang** - Command Line Interface projects and examples built with Go.
- **openai_and_agents** - Experiments with OpenAI API and AI agent implementations in Go.
- **weather_cli** - A weather CLI application fetching data from weather APIs.

## Quick Start

### Run tests in Go with TDD
```bash
cd "Go with TDD"
go test ./...
```

### Explore CLI examples
```bash
cd "CLI w Golang/learning-go-by-examples"
# Browse through various Go example projects
```
